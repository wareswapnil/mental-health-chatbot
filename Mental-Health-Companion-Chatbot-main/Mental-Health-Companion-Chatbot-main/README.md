# 🤖 AI Health Companion

AI-powered personalized Indian diet planner built with Streamlit & Gemini API.

## Features
- Secure Login / Register
- BMI Calculation
- Daily Meal Plan Generator
- Weekly Meal Plan Generator
- Calorie Distribution Chart
- History Storage

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- Matplotlib

## Run Locally

pip install -r requirements.txt
streamlit run app.py
